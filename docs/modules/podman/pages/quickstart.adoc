= Podman Quickstart

Get cc-deck running in a local container in under two minutes.

== Prerequisites

* Podman installed (`podman version`)
* An Anthropic API key or Vertex AI credentials

== Start with API Key

[source,bash]
----
podman run -d --name mydev \
  -e ANTHROPIC_API_KEY=sk-ant-... \
  quay.io/cc-deck/cc-deck-demo:latest

podman exec -it mydev zellij --layout cc-deck
----

== Start with Vertex AI

[source,bash]
----
podman run -d --name mydev \
  -e CLAUDE_CODE_USE_VERTEX=1 \
  -e CLOUD_ML_REGION=us-east5 \
  -e ANTHROPIC_VERTEX_PROJECT_ID=your-project-id \
  -v ~/.config/gcloud:/home/dev/.config/gcloud:ro \
  quay.io/cc-deck/cc-deck-demo:latest

podman exec -it mydev zellij --layout cc-deck
----

== Reconnecting

The container runs in the background with `sleep infinity`.
Reconnect anytime:

[source,bash]
----
# Reconnect to an existing Zellij session
podman exec -it mydev zellij attach

# Or start a new Zellij session
podman exec -it mydev zellij --layout cc-deck

# Quick shell access (no Zellij)
podman exec -it mydev zsh
----

== Stopping

[source,bash]
----
podman stop mydev     # stop the container
podman start mydev    # restart later
podman rm mydev       # remove permanently
----
