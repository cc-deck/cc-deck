= Plugin Overview

cc-deck provides a terminal sidebar that tracks all your Claude Code sessions in real time.
It runs as a narrow panel on the left side of your terminal, showing session status, activity, and providing keyboard-driven navigation.

== The Problem

When you run multiple Claude Code sessions across different tabs, you quickly lose track of what is happening.
Which session is waiting for permission?
Which one just finished?
Which one needs your attention next?
Switching between tabs to check each one is tedious and breaks your flow.

== How cc-deck Solves This

The sidebar gives you a live dashboard of every Claude Code session.
Each session shows its current state: working, waiting for permission, done, or idle.
Smart attend automatically takes you to the session that needs you most.
You never have to manually check each tab again.

Think of it as TweetDeck for Claude Code: multiple columns of activity, all visible at once, with intelligent routing of your attention.

== Key Concepts

Session Dashboard::
The sidebar displays all active Claude Code sessions across all tabs.
Each entry shows the session name (derived from the working directory or git repository), its activity status, and visual indicators for permission requests or completion.

Smart Attend::
A single keypress cycles through sessions that need your attention.
The algorithm prioritizes permission requests first, then completed tasks, then idle sessions.
Sessions that are actively working are skipped.
See xref:attend.adoc[Smart Attend] for the full algorithm.

Navigation Mode::
Toggle a keyboard-driven navigation mode to browse sessions, switch tabs, search, rename, pause, or create new tabs.
See xref:navigation.adoc[Keyboard Navigation] for all shortcuts.

Session Persistence::
Session state survives tab switches and reconnects.
The sidebar tracks sessions across the entire Zellij session, not just the current tab.
