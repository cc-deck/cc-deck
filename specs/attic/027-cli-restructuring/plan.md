# Implementation Plan: CLI Command Restructuring

**Branch**: `027-cli-restructuring` | **Date**: 2026-03-22 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/027-cli-restructuring/spec.md`

## Summary

Restructure the cc-deck CLI to optimize for daily workflows. Six high-frequency environment commands (attach, list, status, start, stop, logs) are promoted to the top level while remaining accessible under `env`. Six legacy Kubernetes-specific commands (deploy, connect, list, delete, logs, sync) and their backing packages are removed. Help output is organized into four named groups (Daily, Session, Environment, Setup) using Cobra's AddGroup mechanism.
