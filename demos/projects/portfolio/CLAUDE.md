# Task: Add dark mode toggle

Add a dark mode toggle button to the portfolio page.

## Requirements

- Add a toggle button in the navigation bar (use a sun/moon emoji or text label)
- Clicking the button should switch between light and dark color schemes
- Dark mode should use a dark background (#1a1a2e) with light text (#e5e7eb)
- Update the CSS custom properties in `:root` and add a `[data-theme="dark"]` selector
- Add a small inline `<script>` at the bottom of the HTML to handle the toggle
- Persist the user's preference using `localStorage`
