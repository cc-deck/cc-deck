## scene:The problem with multiple sessions
You are working on a feature that touches three different services. You fire up Claude Code in each project directory, and within minutes you have three terminal tabs open. One is waiting for your permission to write a file. Another has finished its task. The third is still working. But which is which? You start tabbing through them, checking each one, losing your place, losing your flow. This is the core problem: Claude Code is fantastic in a single session, but as soon as you run multiple sessions in parallel, you lose visibility. There is no dashboard, no overview, no way to know which session needs you right now.

## scene:What cc-deck solves
cc-deck is a Zellij sidebar plugin that gives you a real-time command center for all your Claude Code sessions. Think of it as TweetDeck for Claude Code. Every session appears in the sidebar with a live status indicator. You can see at a glance who is working, who is waiting for permission, who is done, and who is paused. One keypress switches you to the session that needs your attention most. Let us see it in action.

## scene:Launch and start sessions
We launch Zellij with the cc-deck layout. Notice the sidebar on the left. It is empty because no sessions are running yet. Now we start our first Claude Code session in a Python project. Watch the sidebar, it picks up the session within seconds. We open a second tab for a Go project, and a third for a web portfolio. Three sessions running simultaneously, all visible in the sidebar with live status indicators.

## scene:Switching tabs and smart attend
As you switch between tabs manually, notice how the sidebar highlights the current session with a teal background. You always know where you are. But the real power is smart attend. Press Alt+a and cc-deck automatically jumps to the session that needs your attention most. Permission requests come first, then completed tasks, then idle sessions. Press Alt+a again to cycle to the next one. No more hunting through tabs.

## scene:Navigation mode
For more control, press Alt+s to enter navigation mode. An amber cursor appears in the sidebar. Use j and k to move through the list, then press Enter to jump to that session. You can also press g to go to the first session and G to go to the last. Press Escape to leave navigation mode. This is keyboard-driven session switching without taking your hands off the home row.

## scene:Rename and pause
Sessions can be renamed for clarity. In navigation mode, press r and type a new name. You can also right-click a session in the sidebar to rename it. Renamed sessions keep their name across Zellij restarts, so your labels persist. To pause a session, press p in navigation mode. A paused session shows a pause icon and dimmed text. Paused sessions are skipped by smart attend, which is useful when you want to focus on active work and come back to the paused session later.

## scene:Help and new tabs
Press question mark in navigation mode to see all available keyboard shortcuts at a glance. This overlay covers the sidebar until you press question mark again. To start a new session, press n in navigation mode. This creates a fresh Zellij tab where you can launch Claude Code in a new project. The sidebar picks it up automatically.

## scene:Session snapshots
cc-deck can save and restore your session layout. Run cc-deck snapshot save to capture the current set of sessions, their names, working directories, and pause states. Later, run cc-deck snapshot restore to recreate the same layout. This is useful when you want to resume a multi-project workflow after a break or share a session layout with your team.
